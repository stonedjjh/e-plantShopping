# e-plantShopping

Solución al lab de desafío e-plantShopping de IBM

Este es el proyecto final del curso **Desarrollo de aplicaciones frontales con React de IBM** impartido por Coursera. La aplicación simula una tienda en línea de plantas llamada **Paradise Nursery**, donde se gestiona un catálogo de productos y un carrito de compras dinámico utilizando un estado global centralizado.

## Características Principales

* **Catálogo Organizado:** Plantas clasificadas por categorías (purificadoras de aire, aromáticas, medicinales, etc.) con opción de añadir al carrito de forma inmediata.
* **Carrito de Compras Reactivo:** Incremento, decremento y eliminación de productos con actualización automática de cantidades y subtotales por artículo.
* **Cálculos en Tiempo Real:** Barra de navegación con contador global de artículos y cálculo dinámico del monto total de la compra mediante métodos acumuladores.
* **Arquitectura Limpia:** Separación de responsabilidades entre el estado global (Redux Slice) y la interfaz de usuario (Componentes de React), aplicando diseño defensivo en los reducers.
